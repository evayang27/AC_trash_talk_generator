## 垃圾話產生器 
***

### 環境建置與需求
***
+ Node.js
+ Express
+ Express-Handlebars
+ Handlebars-Helpers

### 安裝與執行步驟
1. clone 此專案
```
git clone https://github.com/evayang27/AC_trash_talk_generator.git
```
2. 進入專案資料夾
```
cd path-to-AC_trash_talk_generator
```
3. 初始化
```
npm init
```
4. 安裝 nodemon / express / express-handlebars
```
npm install nodemon express express-handlebars handlebars-helpers
```
5. 啟動程式伺服器
```
nodemon app.js
```
6. 當終端機顯示以下字串，表示成功連結
```
Trash Talk Generator is running on port 3000
```
7. 開啟瀏覽器進入 [http://localhost:3000](http://localhost:3000)
